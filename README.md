     7	## Como editar este código?
     8	
     9	Existem várias formas de editar a aplicação.
    10	
    11	**Use sua IDE preferida**
    12	
    13	If you want to work locally using your own IDE, you can clone this repo and push changes.
    14	
    15	The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
    16	
    17	Follow these steps:
    18	
    19	```sh
    20	# Step 1: Clone the repository using the project's Git URL.
    21	git clone <YOUR_GIT_URL>
    22	
    23	# Step 2: Navigate to the project directory.
    24	cd <YOUR_PROJECT_NAME>
    25	
    26	# Step 3: Install the necessary dependencies.
    27	npm i
    28	
    29	# Step 4: Start the development server with auto-reloading and an instant preview.
    30	npm run dev
    31	```
    32	
    33	**Edit a file directly in GitHub**
    34	
    35	- Navigate to the desired file(s).
    36	- Click the "Edit" button (pencil icon) at the top right of the file view.
    37	- Make your changes and commit the changes.
    38	
    39	**Use GitHub Codespaces**
    40	
    41	- Navigate to the main page of your repository.
    42	- Click on the "Code" button (green button) near the top right.
    43	- Select the "Codespaces" tab.
    44	- Click on "New codespace" to launch a new Codespace environment.
    45	- Edit files directly within the Codespace and commit and push your changes once you're done.
